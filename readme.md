# ChefAid
A web application that allows users to find new recipes based on what ingredient they want to cook with.

## Technologies Used
<ul>
 <li>HTML5</li>
 <li>CSS3</li>
 <li>JavaScript</li>
 <li>jQuery</li>
 <li>Google Fonts</li>
 <li>Font Awesome Icons</li>
 <li>Spoontacular API</li>
</ul>
